## Development

### Clone Repo

```
git clone https://github.com/danstarns/website.git
```

### Install Dependencies

```
npm install
```

### `.env`

In each folder `client` & `server` copy the `.env.example` file to `./.env` and then adjust config to your environment.

### Run Server

```
npm run server
```

### Run Client

```
npm run client
```
